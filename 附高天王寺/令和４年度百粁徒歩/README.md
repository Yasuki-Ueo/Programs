## 令和４年度百粁徒歩について
令和４年度百粁徒歩では初めは第32回百粁徒歩のような滋賀コースを目指していたが、生徒指導部によって潰れてしまった。最終的に高松塚古墳～平城宮跡歴史公園という50.4kmの道のりとなった。十の位を四捨五入したら100kmである。私は特別隊P隊隊長であった。
